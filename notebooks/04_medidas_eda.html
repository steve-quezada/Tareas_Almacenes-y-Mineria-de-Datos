3.3 Medidas de heterogeneidad

3.3.1 Índice de Gini e Índice de Entropía de Shannon

Se analiza la proporción de reportes por entidad para determinar si las desapariciones son un problema generalizado o concentrado en pocos estados.

Code 
def calcular_indices(serie):
    proporciones = serie.value_counts(normalize=True)
    p = proporciones[proporciones > 0]
    n = len(p)
    gini = 1 - np.sum(p**2)
    h_shannon = -np.sum(p * np.log2(p))
    h_max = np.log2(n)
    h_norm = h_shannon / h_max if n > 1 else 0
    return gini, h_shannon, h_norm, n, proporciones

entidades_limpias = df_export_general[df_export_general['ENTIDAD'] != 'SE DESCONOCE']['ENTIDAD']
gini, h_sha, h_norm, n_cat, dist_e = calcular_indices(entidades_limpias)

print(f"--- Heterogeneidad por Entidad ---")
print(f"Categorías: {n_cat}")
print(f"Índice de Gini: {gini:.4f}")
print(f"Entropía de Shannon (H): {h_sha:.4f}")
print(f"Equidad de Pielou (J'): {h_norm:.4f}")

3.4 Medidas de concentración

3.4.1 Índice de Concentración (R) y Curva de Lorenz

Se analiza la concentración de días de retraso entre desaparición y registro.

Code 
def calcular_indice_concentracion(datos):
    x = np.sort(datos)
    N = len(x)
    i = np.arange(1, N + 1)
    fi = i / N
    qi = np.cumsum(x) / np.sum(x)
    numerador = np.sum(fi[:-1] - qi[:-1])
    denominador = np.sum(fi[:-1])
    R = numerador / denominador
    return R, fi, qi

r_valor, fi_vals, qi_vals = calcular_indice_concentracion(dias_validos)

print(f"--- Índice de Concentración (R) ---")
print(f"R = {r_valor:.4f}")
print(f"\nInterpretación: concentración extremadamente alta.")
print(f"Una fracción mínima de los registros acumula la mayor parte del tiempo de retraso.")

## Medidas de heterogeneidad

### Índice de Gini e Índice de Entropía de Shannon

Se analiza la proporción de reportes por entidad para determinar si las desapariciones son un problema generalizado o concentrado en pocos estados.

```{python}
#| label: heterogeneidad
def calcular_indices(serie):
    proporciones = serie.value_counts(normalize=True)
    p = proporciones[proporciones > 0]
    n = len(p)
    gini = 1 - np.sum(p**2)
    h_shannon = -np.sum(p * np.log2(p))
    h_max = np.log2(n)
    h_norm = h_shannon / h_max if n > 1 else 0
    return gini, h_shannon, h_norm, n, proporciones

entidades_limpias = df_export_general[df_export_general['ENTIDAD'] != 'SE DESCONOCE']['ENTIDAD']
gini, h_sha, h_norm, n_cat, dist_e = calcular_indices(entidades_limpias)

print(f"--- Heterogeneidad por Entidad ---")
print(f"Categorías: {n_cat}")
print(f"Índice de Gini: {gini:.4f}")
print(f"Entropía de Shannon (H): {h_sha:.4f}")
print(f"Equidad de Pielou (J'): {h_norm:.4f}")
```

Ambos índices cercanos a 1 confirman una distribución **ampliamente generalizada** de los reportes en todo el país, aunque con picos marcados en ciertos estados.

## Medidas de concentración

### Índice de Concentración (R) y Curva de Lorenz

Se analiza la concentración de días de retraso entre desaparición y registro.

```{python}
#| label: concentracion
def calcular_indice_concentracion(datos):
    x = np.sort(datos)
    N = len(x)
    i = np.arange(1, N + 1)
    fi = i / N
    qi = np.cumsum(x) / np.sum(x)
    numerador = np.sum(fi[:-1] - qi[:-1])
    denominador = np.sum(fi[:-1])
    R = numerador / denominador
    return R, fi, qi

r_valor, fi_vals, qi_vals = calcular_indice_concentracion(dias_validos)

print(f"--- Índice de Concentración (R) ---")
print(f"R = {r_valor:.4f}")
print(f"\nInterpretación: concentración extremadamente alta.")
print(f"Una fracción mínima de los registros acumula la mayor parte del tiempo de retraso.")
```

